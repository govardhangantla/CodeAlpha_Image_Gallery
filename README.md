Lumina Gallery is a premium, high-performance web-based image gallery designed for a seamless visual experience. It features a curated collection of high-quality imagery across **Nature**, **Animals**, and **Technology**, all wrapped in a sleek, modern interface.

---

## ✨ Features

- **🎯 Fluid Filtering**: Instantly switch between categories (Nature, Animals, Tech) with smooth, hardware-accelerated transitions.
- **🖼️ Immersive Lightbox**: View images in full glory with a sophisticated modal featuring navigation controls.
- **📱 Ultra Responsive**: Perfectly optimized for all screen sizes, from mobile devices to 4K desktops.
- **📥 One-Click Downloads**: Easily save your favorite images directly from the lightbox view.
- **⌨️ Keyboard Navigation**: Integrated support for Escape keys and arrow navigation.
- **🎨 Premium Aesthetics**: Custom typography (Outfit), glassmorphism effects, and a refined dark/light balance.

---

## 🛠️ Tech Stack

- **Frontend**: Semantic HTML5 & Vanilla Javascript (ES6+)
- **Styling**: Modern CSS3 (Grid, Flexbox, Variable-based design system)
- **Typography**: [Google Fonts - Outfit](https://fonts.google.com/specimen/Outfit)
- **Deployment**: Integrated with GitHub and ready for Netlify/Vercel.

---

## 🚀 Getting Started

To run this project locally, follow these simple steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/govardhangantla/CodeAlpha_Image_Gallery.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd CodeAlpha_Image_Gallery
   ```

3. **Open the project**:
   Simply open `index.html` in your favorite web browser or use a Live Server extension in VS Code.
